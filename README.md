# Pre-Amplifier-Circuit
# 🎤 Condenser Microphone with Pre-Amplifier Circuit

## 📌 Project Overview
This project demonstrates the design of a **simple condenser microphone circuit with a fixed bias pre-amplifier**, created on a PCB using **EasyEDA**.  
It uses basic components such as a capacitor, resistor, and NPN transistor to amplify weak audio signals captured by a microphone.  
The amplified output is suitable for connection to a power amplifier.

## ⚙️ How It Works
- Acoustic sound signals are converted into small electrical signals by the microphone.  
- A capacitor couples the signal to the transistor’s base, blocking DC and allowing AC.  
- The transistor, configured as a **common-emitter amplifier**, boosts the signal using a fixed bias resistor setup.  
- The output is strong enough to drive a power amplifier for audio applications.
